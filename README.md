# MORPHY COOK OFFICIAL - FRONTEND

Website ini merupakan project frontend pertama yang saya bangun sebagai bagian dari proses belajar pengembangan aplikasi web modern menggunakan ReactJS.

Melalui project ini, saya mempelajari banyak hal, mulai dari pengelolaan state dan custom hooks, integrasi API, handling loading state, autentikasi pengguna (Login, Register, Logout), hingga implementasi Two-Factor Authentication (2FA). Selain itu, saya juga belajar mengenai struktur komponen, manajemen data, serta cara membangun antarmuka yang responsif dan mudah digunakan.

Project ini masih terus dikembangkan dan masih memiliki beberapa fitur yang belum selesai, seperti riwayat aktivitas pengguna, sistem review, serta interaksi antar pengguna. Ke depannya, saya ingin terus meningkatkan kualitas project ini, terutama pada aspek UI/UX, keamanan aplikasi, validasi input, best practice ReactJS, refactoring kode, serta pemisahan logic dan komponen agar lebih terstruktur dan mudah dipelihara.
